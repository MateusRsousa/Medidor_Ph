# Medidor_Ph

Foi implementado um medidor de pH, que mede constantemente o valor de pH de um líquido, e exibe esse valor em um display LCD, juntamente com a data e hora. Foi adicionado ao sistema 3 botões, para que o usuário possa fa zer um registro da medição do valor do Ph, data e hora, consequentemente podendo acessar e vi sualizar esses valores. Sendo que um dos botões faz o salvamento dos valores numa lista, outro botão para acessar os valores da lista e percor rer os valores, e por fim um último botão para apagar os valores da lista.
